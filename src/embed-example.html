Embed your Public Participation Guide by pasting the following HTML code in an HTML embed block in your site:

<div id="idp-embed"></div>
  <script type="text/javascript" src="https://pym.nprapps.org/pym.v1.min.js"></script>
  <script>
    var pymParent = new pym.Parent('idp-embed', 'https://idp.org.za/index.html', {});
  </script>
</div>
